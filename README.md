# Red Pitaya Notes

Notes on the Red Pitaya Open Source Instrument:  https://pavel-demin.github.io/red-pitaya-notes/

Typical framework consist of FPGA bit file, with runs on Red Pitaya programmable logic, a server app which runs on Red Pitaya processing system (CPU), and a client app with runs on a host PC

Bitfile creation example: https://pavel-demin.github.io/red-pitaya-notes/led-blinker/
